# Smart Inventory Tracker

The **Smart Inventory Tracker** is a web-based inventory management system designed to efficiently manage stock, monitor expiry dates, and support logistics optimization. The project uses **C++** for core logic and **Flask (Python)** for the web interface, combining performance with simplicity.
